## Herramientas

Proyecto Realizado en el Lenguaje PHP 7 y FrameWork CodeIgniter 3.

Herramientas Utilizadas en la Creacion:
- HTML5
- CSS3
- Bootstrap
- JavaScript
- AJAX
- Data Tables
- SweetAlert2
- Select2
- JQuery
- MVC
